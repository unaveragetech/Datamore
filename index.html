📋 How to Generate a Dataset

  1. Create a new issue in this repository.
  2. Set the title to the Ollama model name (e.g. Llama 3, Mistral).
  3. Set the body to your topic (e.g. History and dates) or use the structured format:
Topic: World History
Entries: 100
Format: qa
Description: Questions about major world history events

Supported formats: qa (question/answer), facts (fact/explanation), story (prompt/story)
