📋 How to Generate a Dataset

  1. Create a new issue in this repository.
  2. Set the title to the Ollama model name (e.g. Llama 3, Mistral, deepseek-r1).
  3. Set the body to your topic (e.g. History and dates) or use the structured format below.
Topic: World History
Entries: 100
Format: qa
Description: Questions about major world history events

🔢 How Many Entries?

The Entries field controls how many dataset rows are generated (1–500). Recommended starting points:

  • Quick test — Entries: 10 (fast, good for validating a topic)
  • Small dataset — Entries: 50 (default, suitable for fine-tuning experiments)
  • Medium dataset — Entries: 200 (good coverage of a topic)
  • Large dataset — Entries: 500 (maximum; may take several minutes)

Each generated dataset is automatically zipped (containing dataset.json, dataset.jsonl, quality.json, and README.md) and assessed for quality. The ★ score shown on each card is the model’s own overall quality rating (1–10).
